# template-static-site
EJS + Sass + TypeScriptで静的なWebサイトを構築するためのテンプレート 

## Version
Voltaで管理しています。
- Node 22.11.0
- npm 10.9.0

## start
1.  **install**  

    クローンした後に実行してください。

    ```
    npm i
    ```

2.  **develop**

    ```
    npm run dev
    ```

3.  **deploy**

    ```
    npm run build
    ```

## format
```
npm run format:all
```
